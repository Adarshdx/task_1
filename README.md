# Data Cleaning & Visualization Project

## Overview

This project demonstrates data preprocessing and visualization using Python.

The dataset contains student academic records with missing values and outliers. Data cleaning techniques are applied before generating visual insights.

## Features

- Handle missing values
- Remove outliers
- Remove duplicate records
- Generate visual reports
- Data storytelling through charts

## Technologies Used

- Python
- Pandas
- Matplotlib
